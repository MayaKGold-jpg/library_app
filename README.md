# Library Book Checkout Program

## How to Run
To run this program, use:

cargo run


## What It Does
This Rust program simulates a simple library system. It allows you to:

- Add books to a library using a vector (`Vec<Book>`)
- List all books in the library
- Check out a book by title
- Track whether a book is available or checked out

## Features
- Uses a `Book` struct with title, author, and checkout status
- Demonstrates Rust ownership and borrowing
- Uses mutable and immutable references correctly
- Updates book status when checked out

## Example Output
The program prints the list of books, checks out one book, and then prints the updated list.
